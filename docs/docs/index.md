# Welcome to Bensemble 😎

**Bensemble** is a lightweight, modular library for Bayesian Deep Learning and Neural Network Ensembling in PyTorch.

## Why Bensemble?

- **Unified API**: One interface for MC Dropout, NAS, and Variational Inference.
- **Uncertainty Analytics**: Built-in tools to separate Aleatoric and Epistemic uncertainty.
- **Calibration**: Easy-to-use Temperature and Vector scaling.
